Basic Tags

Tag Name Description
<html> HTML Creates an HTML document
<head> Header Sets off the title & other info that isn't displayed
<body> Body Sets off the visible portion of the document
<title> Title Puts name of the document in the title bar; when bookmarking pages, this is what is bookmarked

Text Tags

Tag Name Description
<h1> <h6> HTML Creates headlines -- H1=largest, H6=smallest
<strong> Bold Creates bold text
<em> Italic Creates italicised text

Link Tags

Tag Name Description
<a>href="URL">clickable text</a> External Hyperlink Creates a hyperlink to a Uniform Resource Locator
<a>href="mailto:@EMAIL">text</a> Email Hyperlink Creates a hyperlink to an email address
<a>href="#NAME">text</a> Internal Hyperlink Creates a link to that target location

Formatting Tags

Tag Name Description
<p> Paragraph Creates a new paragraph
<br> Line Break Inserts a line break
<blockquote> Blockquote Puts content in a quote - indents text from both sides
<div> Divider Used to format block content with CSS
<span> Inline content Used to format inline content with CSS

List Tags

Tag Name Description
<ul> Unordered List Creates an unordered list
<> Ordered List Creates an ordered list
<li> List Item Encompasses each list item
